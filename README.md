# 每日热量计算器

一个使用 **React + Vite** 开发的中文热量记录工具，支持记录食物名称、重量、热量、蛋白质、碳水、脂肪，并自动统计今日总摄入。数据保存在浏览器 `localStorage`。

## 功能

- 中文界面，移动端优先适配
- 输入食物名称、重量、热量
- 输入三大营养素：蛋白质 / 碳水 / 脂肪
- 自动汇总今日总热量和三大营养素
- 数据本地持久化（localStorage）

## 本地运行

1. 安装依赖

```bash
npm install
```

2. 启动开发环境

```bash
npm run dev
```

3. 打开浏览器访问（默认）：

```text
http://localhost:3000
```

## 打包

```bash
npm run build
```
